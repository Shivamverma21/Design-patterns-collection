# Low Level Design Patterns Collection

This repository contains implementations of various low-level design patterns in Java.

## Patterns Included:
- Singleton
- new line Items in progess...

## Setup Instructions:
1. Clone the repository.
2. Use Maven to build the project: `mvn clean install`.
3. Run the application to see the design patterns in action.

## Contributing:
Feel free to create a pull request if you want to contribute to this repository! You can create examples of any design patterns and place them in their respective folders.

Thank you for contributing!
